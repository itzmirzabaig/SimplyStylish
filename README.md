# Simply Stylish

A chic affiliate marketing site showcasing accessories, footwear, perfumes, and furniture.  
Built with vanilla HTML, CSS, and JavaScript, and deployed via GitHub Pages.

## Pages

- **Home** (`index.html`) – Welcome & Newsletter sign-up  
- **About Me** (`about.html`) – My story & Ivy League photo slider  
- **Explore** (`explore.html`) – Product categories & “Coming Soon” cards  
- **Contact** (`contact.html`) – Contact details
